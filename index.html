Tux (space) invader

Tux invader is an adaptation of the very famous game "Space Invader", a game where you control a ship and you have to kill enemies that shoot on you. To code this, we use c++, on QT Creator, and we use the programming interfaces SFML and MinGL. On this project, we were four on this project, beause it was a school project and we had around one month and an half to make it. There is still some defauts and bug, but we made some good options. For exemple, we can stores the ten best scores, which are visible in a menu a the beginning of the game, and there is an infinity of levels for the best player.
